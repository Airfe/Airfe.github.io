React（一）基础知识

React入门

介绍

用于动态构建用户界面的 JavaScript 库(只关注于视图)。

  • 英文官网: https://reactjs.org/
  • 中文官网: https://react.docschina.org/

特点：

  • 声明式编码、组件化编码
  • React Native （可以使得js编写安卓、ios应用）编写原生应用
  • 高效（优秀的Diffing算法）

React JSX

  1. 本质：React 定义的一种类似于XML 的 JS 扩展语法：JS+XML。本质上是React.createElement(component,props,...children)方法的语法糖。

  2. 作用：简化创建虚拟DOM

不是字符串，也不是html/xml标签，最终产生的就是一个js对象

可以看出jsx写起来更易读，虽然他的底层也是转化为了js的写法，但是更加方便了书写。

  1. 特点

  • 定义虚拟DOM，不要写引号“”，因为不是字符串

  • 标签中混入JS表达式的时候使用{} ，{}里面放的是js表达式，即有返回值的js语句。（if条件、for循环不是js表达式）

  • 样式的类名指定不要使用class，使用className

  • 内联样式要使用双大括号包裹，即style = {{key: value}}的形式

    • 外面那层表示表达式，里面那层表示对象

  • 不能有多个根标签，只能有一个根标签

  • 每个标签必须闭合

  • 标签首字母，其实就是createElement的第一个参数type的要求。

    • 小写字母开头，必须是html同名元素，如果html中无该标签对应的元素，报错；
    • 大写字母开头，react会去渲染对应的组件，如果没有就报错。

jsx允许在{}放数组，会自动展开，但对象不行。
