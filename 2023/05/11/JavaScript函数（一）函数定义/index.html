函数声明与函数表达式

JavaScript 引擎在加载数据时对这两种方式是区别对待的。

  • 函数声明的关键特点是函数声明提升，即函数声明会在代码执行之前获得定义。

    这意味着函数声明可以出现在调用它的代码之后：

  • 而函数表达式必须等到代码执行到它那一行，才会在执行上下文中生成函数定义。

除了函数什么时候真正有定义这个区别之外，这两种语法是等价的。

箭头函数

很大程度上，箭头函数实例化的函数对象与正式的函数表达式创建的函数对象行为是相同的。任何可以使用函数表达式的地方，都可以使用箭头函数：

参数：如果只有一个参数，那也可以不用括号；但没有参数，或者多个参数的情况下，必须使用括号。

函数体：箭头函数体如果只有一行代码，可以不用大括号包裹。而且，省略大括号会隐式返回这行代码的值

箭头函数虽然语法简洁，但也有很多场合不适用。

箭头函数不能使用 arguments、super 和 new.target，也不能用作构造函数。此外，箭头函数也没有 prototype 属性。
